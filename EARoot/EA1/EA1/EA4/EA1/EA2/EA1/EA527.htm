Las matrices muestran las relaciones principales entre entidades de modelo relacionadas.<br/><br/>Las matrices constituyen la materia prima para el desarrollo de puntos de vista y también actúan como un recurso clave para la evaluación de impacto, llevada a cabo como parte del análisis de brechas.<br/><br/>Las siguientes matrices deben ser consideradas para el desarrollo dentro de una Arquitectura de Negocios:<br/><br/>Matriz de flujo de valor/Capacidad (muestra las capacidades necesarias para soportar cada etapa de un flujo de valor)<br/>Matriz de estrategia/capacidad (muestra las capacidades necesarias para admitir declaraciones de estrategia específicas)<br/>Matriz de capacidad/organización (muestra los elementos de la organización que implementan cada capacidad)<br/>Matriz de interacción del negocio (muestra la dependencia y la comunicación entre organizaciones y actores)<br/>Matriz actor/rol (que muestra los roles asumidos por cada actor)<br/>La estructura de las matrices se basa en los atributos de las entidades metamodelo, tal como se definen en la <i>Parte IV, 30. Metamodelo de contenido</i>.<br/>
